#  Data Analytics Data Cleaning & Preparation P1

##  Overview
This project focuses on **data cleaning and exploratory analysis using Excel**.  
The dataset contains **customer orders, products, payments, shipping details, and promotions**.  
The objective is to prepare the raw dataset, identify key patterns, and document insights without building dashboards.

##  Dataset Details
- **OrderID** – Unique identifier for each order  
- **Date** – Order date (numeric format)  
- **CustomerID** – Unique customer reference  
- **Product** – Product category (Monitor, Phone, Tablet, Chair, Desk, Printer, Laptop)  
- **Quantity & UnitPrice** – Order details for calculating revenue  
- **PaymentMethod** – Cash, Debit Card, Credit Card, Online, Gift Card  
- **OrderStatus** – Shipped, Delivered, Cancelled, Returned, Pending  
- **CouponCode & ReferralSource** – Promotions and marketing channels  
- **TotalPrice** – Final order value  

##  Project Work
- **Data Cleaning**
  - Removed duplicates and corrected inconsistent values  
  - Standardized numeric formats (UnitPrice, TotalPrice)  
  - Converted dates into readable format  
  - Handled missing or invalid entries  

- **Data Preparation**
  - Organized data into structured tables  
  - Added calculated fields (e.g., revenue per product)  
  - Applied filters and sorting for analysis  
  - Documented all steps for reproducibility  

- **Analysis**
  - Sales by product category  
  - Payment method distribution  
  - Order status breakdown  
  - Impact of coupon codes and referral sources on sales  
  - Customer purchase behavior (items in cart, average spend)

##  Key Insights
- **Phones, Tablets, and Laptops** generate the highest revenue.  
- **Cash & Credit Card** are the most common payment methods.  
- **Cancelled & Returned orders** highlight potential issues in product quality or delivery.  
- **Coupon codes (SAVE10, FREESHIP, WINTER15)** significantly influence purchase decisions.  
- **Referral sources like Instagram & Facebook** drive strong customer engagement.
